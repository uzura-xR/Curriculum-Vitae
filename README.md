### 2022年　1月~2月:靴検品VR
#### プロジェクト概要
VRプロフェッショナルアカデミー受講時の開発
作品名『靴検品VR』https://youtu.be/jbHwdx8v63c
* このコンテンツは靴販売の現場や出荷作業の現場で起こりえる、靴の品番間違い、サイズ間違い、左右の入れ間違いなどを、なくすためVR学習作品です。
#### 開発スケジュール
* 2か月
#### 担当内容
* 一人ですべて作成
#### 実績・習得スキル
自作したScriptの一例
・Text送りと音声を連携させて動かすためのScript　https://gist.github.com/uzura-xR/0617905284308f4f9ebb675096debad9

VRフェスにて『体験デザイン賞』受賞(10期生受賞作品欄に記載/下記のリンク先の上から三段目に記載されています)
https://vracademy.jp/introduction/

* 誰でもわかりやすい『チュートリアル』と『このコンテンツをする意味』をいかに初めて体験する人にも感じてもらえるかを軸に作成
* VRoidStudioで自作アバター2体作成
* UniVRMを使用してunityにアバターを反映
* 音声に合わせてリップシンクするようにOVRLipSyncの機能などを利用し音声の波形に合わせてキャラクターの口が動くように実装
* Mixamoから使用するAnimationを収集し実装
* 電話機（blenderで作成）
* カミナリのエフェクト（VFXshader GraphとVFX Effect Graphで作成）
* コンテンツはフルボイスになっていて私、自らTwitterで募集をし声優オーディションを開催して配役を決定
* 人に協力を求めて良いものを作ることの難しさや面白さを知るいい経験になった。

私自身が、靴職人、靴デザイナー、靴販売・物流業務など靴に関わる仕事を一通り経験してきた経験から靴業界で実際に起きていた問題をピックアップしこのコンテンツを作成しました。

このコンテンツに取り組むことは、私自身が教育系のVRシュミレーションに興味があったからなのですが、企業が抱える現場の問題、特に研修のビデオなどだけでは伝わらないこと、実際に体験してみないとわかりにくいことなど、『なぜこれを覚えることが重要なのか』という点を重視して作りました。

このコンテンツでは、**誰でもわかりやすい『チュートリアル』と『このコンテンツをする意味』をいかに初めて体験する人にも感じてもらえるかを大事に作成**したのでＶＲフェスにて『体験デザイン賞』をいただけたことは狙い通りにもなり大変うれしかったです。

習得スキルについては、今までＶＲアカデミー内で学んだことはもちろんですが**VRoidStudioで自作アバター
2体作成しUniVRMを使用してunityにアバターを反映、音声に合わせてリップシンクするようにOVRLipSyncの機能などを利用し音声の波形に合わせてキャラクターの口が動くように実装**しています。

キャラクターの**アニメーションはMixamoから使用するAnimationを収集し実装**しています。

コンテンツの性質上教育系のシュミレーションのため繰り返し作業や単調な説明などが続き、体験者が飽きてしまう可能性があったため、飽きさせず、コンテンツに集中かつ楽しんでもらえるよう、キャラクターの動きや出現位置、その他、**電話機（blenderで作成**）や怒られてしまったイメージを強くし責任を感じてもらいやすくするために**自作したカミナリのエフェクト（VFXshader GraphとVFX Effect Graphで作成）などの細かなギミックの制作**に力を入れました。

また、**今回のコンテンツはフルボイスになっていて私、自らTwitterで募集をし声優オーディションを開催して配役を決定**しています。

結果として、コンテンツの内容がさらにわかりやすくなったこと、そして人が話していることで、よりシナリオに没入できる内容になったと思います。

また、**オーディションを開催したことで、人に協力を求めて良いものを作ることの難しさや面白さを知るいい経験になりました。**

## 職務経歴

### 2015年4月~2017年4月:株式会社プープディドゥ
#### プロジェクト概要
* 靴のデザイン企画
* 年2回行われる展示会のためのシューズ企画デザインを担当
#### 担当フェーズ
* 男性用のカジュアルシューズ部門を担当
#### 業務内容
* デザインの作成はもちろん使う木型、素材の選定と海外発注する荷物のサンプル管理やインボイスの作成なども行ってきました。
* 上がってきたサンプルに対して修正lineを入れて再度修正依頼などを行ってきました。
#### 実績・習得スキル
* 百貨店に**自分のデザインした靴が商品化**され販売されました。

  靴の展示会は各百貨店や専門店様がお越しになり、商品化してほしいシューズをセレクトしオーダーされていくシステムで、展示されたシューズ自体を気に入ってもらうことはもちろんですが、一つの百貨店や専門店のバイヤーにだけ注文されても商品化することはできません。

  複数の店舗から注文があって初めて商品化やカラーバリエーションも増やして大量生産に回せるので、バイヤーさんに商品説明をする際、営業担当の方に事前にシューズのアピールポイントやこの靴はどんな人をターゲットにしてデザインしているなどの細かな詳細を伝える必要があります。

  この経験から、**世の中に求められる商品はどんなものなのかを決める際の物差しとなるターゲットの決め方やマーケティングを学ぶことができ、さらには靴職人さんとのヒヤリングと営業担当への伝達などのスキルが身に付きました。**

### 2017年4月~2019年4月:株式会社プープディドゥ

* スタッフ教育
#### 担当フェーズ
* 教育スタッフ→店長
