## 自己PR
私は興味を持ったものに対して調べものをしたり学習することに対して努力は惜しまない性格だと思います。
その証拠に、2020年にVRゲームであるアルトデウスB.CをプレイしてVRの世界にのめり込みました。気が付くと3日後にはVRの勉強ができるスクールであるVRアカデミーに通うことを決めて入学手続きをしていました。

その際、授業が始まるまで3か月の期間があり、とにかくVRの勉強をしてみたいとひたすらに調べ、2021年4/29～5/5の7日間にわたって開催された「バーチャル空間(VR)のインディゲーム系展示会であるGameVketZero」に出展することを決意して実際に3か月には出展することに成功しました。

この時点では全くunityも触ったことがなかったので、わからないことばかりでしたが、作ってみたいやってみたいという気持ちの方が断然高かったため、検索力のみを頼りに完成させました。VRアカデミーに入ってからは、unityの基礎から学び、プログラミングも覚えてきました。

その中で自分が思い描いたことが作品として形になっていくことにわくわくして常に面白さを感じていたので、この1年間、仕事終わりはほぼ毎日のようにunityや3DCG関係の勉強をする日々を過ごしていました。

学習していく中で、はじめて体験したVRゲームであるアルトデウスを再度プレイし、どのような作りになっていて、これは自分の作品でも再現してみたいと思う部分がたくさんあり、自分が作るVR作品はかなり貴社の影響を受けて作成しています。

また、多くのVRゲームや作品を体験していく中であることに気が付きました。

それはUIのデザインや説明文が小さすぎたり、音声での説明がなかったりといった、『はじめてやる人が理解しにくい』コンテンツが意外と多いということです。

アルトデウスではそういった『何をしていいかわからない』という迷いが一切なかったと感じています。
それゆえに、私の中で没入感を非常に感じられた作品でした。

アルトデウスを体験した事でコンテンツを作成する上で、わかりやすさとはVRにおいて非常に大事なのだということを学べました。

また、私自身がアルトデウスのロボットシークエンスが非常に大好きなこともありますが、UI周りに触れたら『ちょっとした動き』があったり『気持ちのいい効果音』が入っていたりとコンテンツ自体に没入させる細かな作りが、素敵だな思ってます。

VRを１年間勉強してきたからこそ、作り手の作品に対する細かな気配りに感動しました。

そういった影響もあり、VRアカデミーに在学中に私が1番注力したことはわかりやすいVRを作る事でした。

現職でライター業務を行うことも多いので、コンテンツ内でユーザーにどう伝えればわかりやすいかシナリオを作ったり、画像編集ソフトを使ってオリジナルのUIを作るなど工夫をこらし、最終的に作成した靴検品VRというコンテンツではフルボイスで音声を収録するために、Twitterで声優募集を行いオーディションをして配役を決めるなど、わかりやすいコンテンツ作りをするために行動しました。

結果として体験デザイン賞という賞を頂くことが出来ました。

今後この業界の成長事態にも、とても興味があるので自ら情報収集も行うようにしています。
VRだけでなくARやMRの分野も気になっています。

学習面ではまだまだ足りない知識が多いかと思いますが、持ち前の好奇心でたくさん知って習得していきたいと思っています。この業界に入りたいからこそ勉強してきたのでよろしくお願い致します。
